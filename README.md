# Navgrah by Raani

Website for Navgrah by Raani, built with Next.js 14, React 18, TypeScript, Tailwind CSS, Framer Motion, Lucide icons, React Hook Form, and Zod.

## Getting Started

```bash
cd /d/raani
nvm use 20
npm install
npm run dev
```

Open `http://localhost:3000`.

See `SETUP.md` for the Git Bash Node environment setup and `requirements.md` for the dependency list.
